The Design Token System

Every color, type style, spacing value, shadow, border radius, and motion curve in Vendoora — codified as CSS variables. Three-tier architecture: primitives, semantic aliases, and component-level tokens. This is the technical foundation the engineering build points at.

Border Radius

Direction A leans toward slightly-rounded corners — sharp enough to feel institutional, soft enough to feel human. Buttons use radius-lg (8px), cards use radius-2xl (16px), pills use radius-full.

--radius-sm4px · inputs
--radius-lg8px · buttons

Step 5 of 7

Shadows / Elevation

Six levels of elevation. Direction A uses subtle shadows — no dramatic drop shadows or glow effects. Each shadow gets darker AND further offset as elevation increases, mimicking real light.

shadow-xsHairline lift
shadow-smButtons, inputs
shadow-mdCards default

shadow-xlModals
shadow-2xlOverlays, dialogs
Step 6 of 7

Motion

Five duration tokens, four easing curves. Direction A uses fast, restrained motion — 120-200ms for most state changes, longer durations only for full-page transitions.

--duration-fast120ms · button states, hover
--duration-default200ms · most UI transitions
--duration-slow320ms · modal open/close

The Token CSS — Copy Into Your Codebase

Save this file as tokens.css in your styles folder. Import it once at the root of every product surface. The tokens cascade everywhere.

/* ======================================== VENDOORA DESIGN TOKENS — tokens.css v1.0 · May 22, 2026 Import once at the root of every surface. ======================================== */ :root { /* COLOR PRIMITIVES */ --blue-700: #1A3DAE; /* Brand primary */ --blue-800: #142E7A; /* Brand primary deep */ --red-500: #CE1126; /* Liberian accent */ --neutral-0: #FFFFFF; --neutral-900: #0A1226; --success-500: #047857; /* TYPOGRAPHY */ --font-sans: 'Inter Tight', system-ui, sans-serif; --font-serif: 'Fraunces', Georgia, serif; /* SPACING (4px scale) */ --space-4: 16px; --space-6: 24px; --space-8: 32px; /* RADIUS */ --radius-lg: 8px; /* Buttons */ --radius-2xl: 16px; /* Cards */ --radius-full: 9999px; /* Pills */ /* SHADOWS */ --shadow-md: 0 4px 12px -2px rgba(10, 18, 38, 0.08); /* MOTION */ --duration-default: 200ms; --easing-ease: cubic-bezier(0.4, 0, 0.2, 1); /* SEMANTIC ALIASES */ --color-action-primary: var(--blue-700); --color-text-default: var(--neutral-900); --color-surface-default: var(--neutral-0); --color-verified: var(--success-500); --color-escrow: var(--blue-700); --color-delivery-code: var(--red-500); /* Full file in your /styles/tokens.css — this is an excerpt */ }

The full token file (visible by viewing this HTML's <style> block, top of the file) has the complete primitive + semantic + component layers. When the engineering build starts with Claude Code, paste this into the project's root stylesheet and every screen built afterward inherits the system automatically.
